Title: El Buvette

Description:

In a strange parallel universe inside FST, Wadi3 runs the legendary campus snack bar “El Buvette.” His menu is… minimal. You’ll only find Tropico, Prince, Soufflet, Eau, and Capucin.
To make life easier for students, he built a small website where customers can check item prices. Simple, right?
Well… not quite.
Rumor has it Wadi3 reused the same database for something much more important — a secret key that can open the portal back to the real world!
If that key is hidden in the database, maybe the website can help you reach it…
Can you dig deep enough to escape this universe?
